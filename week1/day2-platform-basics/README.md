# Salesforce Platform Basics

## Introduction

Salesforce is a cloud-based platform used by companies to manage customer relationships, business processes, sales, and services. It helps organizations store data, automate tasks, and create business applications efficiently.

The Salesforce Platform provides tools for both low-code and coding-based development. Businesses can customize applications, create objects, automate workflows, and improve productivity using Salesforce.

---

# What is an App?

An App in Salesforce is a collection of tabs, objects, and tools grouped together for a specific purpose.

Examples:
- Sales App
- Service App
- Marketing App

Apps help users organize their work and access related features easily.

---

# What is an Object?

An Object is used to store data in Salesforce.

Objects are similar to database tables.

Examples:
- Account
- Contact
- Opportunity

Objects contain:
- Fields
- Records

Salesforce provides:
- Standard Objects
- Custom Objects

---

# What is a Tab?

A Tab is used to access objects and records in Salesforce.

Tabs appear in the navigation bar and help users open different sections quickly.

Examples:
- Accounts Tab
- Contacts Tab
- Opportunities Tab

---

# Configuration vs Coding

## Configuration

Configuration means creating features without writing code.

Examples:
- Creating objects
- Creating fields
- Page layouts
- Reports
- Flows

This is mainly done by Salesforce Admins.

---

## Coding

Coding means building advanced functionality using programming languages.

Salesforce Developers use:
- Apex
- Lightning Web Components (LWC)
- APIs

Coding is used for:
- Complex automation
- Custom UI
- Integrations
- Advanced business logic
